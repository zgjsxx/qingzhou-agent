# xu-agent v1.1.0

- 新增子 Agent 工具：支持独立上下文执行复杂子任务，默认隔离内部工具流并提供可选调试开关
- 新增 HTTP MCP 工具接入：启动时读取 MCP 配置并将 tools/list 发现的外部工具注入主 Agent
- 新增插件面板：前端左侧可查看当前配置的 MCP 插件
- 新增后台任务取消：支持通过 background task id 主动停止正在运行的后台 shell 任务
- 新增文件下载链接：Agent 生成结果文件时自动在回复中提供可点击的下载链接
- 新增 snip_compact 上下文裁剪：消息数超过阈值时自动裁掉中间旧对话，保留首尾关键消息
- 新增会话级 cron scheduler：支持创建、查看、取消定时任务，到点后自动向原会话注入提示词并执行
- 新增 LlamaIndex RAG：支持构建本地文档索引、通过 rag_search 检索证据片段，并提供手动 build index 脚本
- 新增飞书长连接机器人接入：支持本地后端主动连接飞书事件流，接收文本消息并调用 agent 回复
- 调整项目技能目录：默认从 `backend/skills` 加载技能，前端技能面板与后端加载路径保持一致
- 修复 TypedDict 兼容性：使用 typing_extensions 替代 typing.TypedDict，消除 Python < 3.12 warning
- 新增 Anthropic Bearer 认证：支持通过 `LLM_AUTH_TOKEN` 或 `ANTHROPIC_AUTH_TOKEN` 配置主模型及上下文摘要模型，同时保留 API Key 认证方式
