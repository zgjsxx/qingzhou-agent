# xu-agent v1.1.0

## 新功能

### 文件下载链接

Agent 生成结果文件（CSV、PDF、脚本等）时，现在会在回复中提供可点击的下载链接，用户无需手动查找后端文件路径即可直接下载。

- **后端**：系统 prompt 新增下载链接规则，指导 Agent 在生成文件后自动提供 `[下载 文件名](前端URL/api/local/downloads/相对路径)` 格式的链接
- **前端**：新增 `/api/local/downloads/[...path]` API 路由，安全地代理后端目录的文件下载
  - 路径穿越防护：使用 `path.relative` 判断替代 `startsWith`，避免前缀误判风险
  - 中文文件名支持：`Content-Disposition` 同时提供 `filename`（ASCII fallback）和 `filename*=UTF-8''` 编码
  - 文件大小限制：最大 100MB
- **前端 rewrite**：`/output/:path*` 自动重写到 `/api/local/downloads/output/:path*`，兼容 LLM 生成的非标准 URL 格式
- **环境变量**：新增 `FRONTEND_URL` 配置项，用于生成下载链接的前端基础 URL

## 配置变更

- `backend/.env.example` 新增 `FRONTEND_URL=http://localhost:3000`
- `frontend/next.config.mjs` 新增 `/output/:path*` → `/api/local/downloads/output/:path*` rewrite 规则
