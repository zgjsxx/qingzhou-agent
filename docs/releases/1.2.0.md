# xu-agent v1.2.0

- 将核心 agent 运行时代码从 ackend/src/agent_*.py 与 ackend/src/llm_config.py 迁移到根目录 gent/ 包；ackend/src/agent.py 保留为 LangGraph 兼容入口，	ools.py、skills.py 等后端工具模块暂时留在 ackend/src/，降低本阶段重构风险。

- 重构 gateway 平台接入代码目录：飞书、QQ、微信、Telegram、Discord 桥接模块迁移到 `gateway/platforms/`，入口与测试改为通过统一 gateway 包引用，为后续继续拆分后端目录做准备。
- 将前端工程目录从 `frontend/` 迁移为 `web/`，并同步更新启动、构建脚本和相关文档路径，作为根目录扁平化重构的第一步。
- 将 gateway 桥接源码从 `backend/src/gateway/` 提升到根目录 `gateway/`，运行态数据路径暂时仍指向 `backend/`，继续保持后端其它模块不变。

## 最新补充

- 新增 Discord 官方 Gateway 机器人桥接：支持私聊和服务器频道文本、用户白名单、频道提及控制、连续消息合并、附件下载、统一 slash command，以及按 channel 隔离的历史和中间 AI 文本转发。

## 版本说明

1.2.0 进入开发阶段。本文件用于持续记录该版本新增功能、行为变更、问题修复和升级注意事项。

## 变更记录

- 将项目根版本、后端 Python 包、前端应用及 MCP 客户端版本统一更新为 `1.2.0`。
- 将前端字体从构建时下载的 Google Inter 改为项目内置 Geist 字体，支持完全离线构建。
- 优化本地启动顺序：等待 LangGraph `/info` 就绪后再启动前端，并为前端连接检查增加短暂重试，避免启动竞态产生误报。
- 后端启动默认禁用自动打开 LangSmith Studio 浏览器页面，需要时仍可手动访问。
- 前端 Plugins 面板支持启用/禁用已配置的 MCP 插件，并提示需要重启后端后生效。
- 修复 `run_ssh_command` 在远端命令读取 stdin 时容易挂起直到超时的问题：SSH 调用改为显式非交互执行，关闭 stdin 转发，并在超时时清理进程树、返回已捕获输出。
- 新增 `ssh_upload_file` 和 `ssh_download_file` 工具：支持通过 SSH/SFTP 在本地与远程主机之间上传、下载文件，复用已有 SSH 主机、端口、用户、密码和密钥配置，并接入统一的工具权限审批与调用签名缓存。
- 修复前端左侧本地面板中“插件”“运行监控”菜单项的中文乱码问题，并在源码中保留编码规避说明。
- 新增 `/compact [focus]` 手动上下文压缩命令：命令由后端上下文中间件拦截，不交给主模型执行，并复用自动压缩的摘要、消息替换、metadata 与日志更新逻辑；手动压缩默认尽量把历史消息都压入摘要，可通过 `AGENT_MANUAL_COMPACT_KEEP_MESSAGES` 调整保留的最近原文数量。
- 新增 `/help` 斜杠命令：后端直接返回确定性的命令清单，不交给主模型生成，避免帮助内容和实际支持的命令不一致。
- 统一斜杠命令入口：`agent_commands.py` 负责解析 `/help`、`/clear`、`/compact`，自有 UI、飞书和后端图中间件复用同一套命令模型，减少新增命令时的分散改动。
- 新增基于 `qq-botpy` 的腾讯 QQ 机器人桥接层：支持频道 `@机器人` 与私信文本消息接入，复用现有 LangGraph graph、历史消息、日志与 slash command 分发，并预留群 `@` / C2C 事件处理入口。
- 扩展飞书消息桥接能力：支持接收并下载用户发送的文件和图片，保留中文文件名并将本地文件信息交给 Agent；支持在可配置等待窗口内合并连续发送的文件、图片和文本消息后统一提交给模型；reaction 请求改为异步执行，并在最终回复完成后按消息准确清理，避免网络延迟打断消息合并或提前移除处理中状态。
- 新增 Tavily Web Search MCP 配置：Plugins 页面可配置 API Key、默认结果数、搜索深度和启用状态，自动创建本地远程 MCP 配置；配置 API 仅返回凭据是否存在，不回传密钥，重启后端后即可加载 Tavily 的搜索、提取、站点地图和爬取工具。
- 新增微信 iLink 单聊文本桥接：支持二维码登录并在本地保存机器人凭据，通过长轮询接收微信私聊消息，持久化同步游标和会话 `context_token`，复用统一 slash command 与 LangGraph 对话，并提供消息去重、分段回复和自动退避重试。
- 新增 Telegram 官方 Bot API 桥接：支持私聊和群聊文本、用户白名单、群聊提及控制、连续消息合并、图片与文档下载、统一 slash command，以及按 chat 隔离的历史和中间 AI 文本转发。
- 新增 `matplotlib-charts` 图表技能：支持通过统一 JSON 配置生成柱状图、饼图和折线图，提供多数据系列、中文字体、数值标签、输入校验与 PNG 输出，并将 `matplotlib` 加入后端依赖。
