# 基本设备表的定义

## other_device

### 表基本信息
该表用于存储基本设备的基础信息、类型属性、状态信息、继电保护关联信息、光伏相关参数、容量信息及设备模型关联等内容。

### 主键字段
- other_id:基本设备的id，序号从432000001开始

### 唯一字段
- other_alias:基本设备别名
**别名格式（硬约束）**: 必须为四段，格式为 `{fac_alias}.Farm.{bay_name}.{device_id}`，
且前 3 段必须精确匹配 bay_info 表中的一条 bays_alias 记录。
第二段固定为 `Farm`。
数据库有触发器 ti_other_device 校验此规则，不满足格式会拒绝写入。
例如 `LENAN.Farm.Storage.DEFINE`、`test_bess_002.Farm.bay001.dev001`。
