# 间隔相关表的定义

## bay_info

### 表基本信息
该表用于存储间隔（bay）的基础信息、类型属性、状态标识、扩展配置、关联关系及模板/图形资源等内容。

### 主键字段
- bay_id:间隔id，序号从430000001开始

### 唯一字段
- bay_alias:间隔别名
**别名格式（硬约束）**: 必须为三段，格式为 `{fac_alias}.Farm.{bay_name}`。
例如 `LENAN.Farm.Storage`、`test_bess_002.Farm.bay001`。
第二段固定为 `Farm`，第一段为所属厂站的别名，第三段为自定义间隔名。
